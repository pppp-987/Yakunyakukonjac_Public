更新方法
    1 . プロジェクトフォルダへの移動
    cd \venv_YakunyakuKonjac\YakunyakuKonjac_Public
    2 . gitからダウンロード
    git pull origin
    3 . ログの確認
    git log
        "バリデーションチェック機能の追加" という文章が表示されてたら正常


コンフリクト(競合発生時)
    以下のコードを実行
        git reset --hard HEAD
        最終コミット前まで戻す　ファイル変更内容もすべて戻す
        変更箇所がある場合は別の場所に保存してください
